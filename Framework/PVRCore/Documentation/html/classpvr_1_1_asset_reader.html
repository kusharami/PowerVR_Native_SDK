Detailed Description

template<typename AssetType>
class pvr::AssetReader< AssetType >

Base class for an AssetReader, a class that can read Assets from a provided Stream.

Template Parameters
AssetTypeThe typename (class) of assets that will be read from this AssetReader.

A reader of a specific type of assets will normally inherit from an AssetReader of the specified AssetType. For example, a reader that reads Models from POD files will be: class PODReader:AssetReader<Model>
