Detailed Description

Add blending configuration for a color attachment. Some API's only support one blending state for all attachments, in which case the 1st such configuration will be used for all.

— Defaults — Blend Enabled:false, Source blend Color factor: false, Destination blend Color factor: Zero, Source blend Alpha factor: Zero, Destination blending Alpha factor :Zero, Blending operation color: Add, Blending operation alpha: Add, Channel writing mask: All
