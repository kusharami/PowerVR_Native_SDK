Detailed Description

Interface for the Platform context. Contains all the operations that the pvr::Shell and StateMachine classes needs in order to be able to function (creation, destrurction, create a window, swap the buffers, and make current). Specific platform contexts (Egl, Vulkan) implement this interface.
