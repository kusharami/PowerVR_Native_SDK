PlatformContextThe platform context is the class wrapping all platform-specific objects required to power the PVRApi Graphics context (Displays, windows, configurations etc.).
eglThis class's static members are function pointers to the EGL functions. The Shell kicks off their initialisation on before context creation. Use the class name like a namespace: egl::ChooseConfig.
eglextThis class contains function pointers for all EGL extensions that existed at the time of publishing. Can be updated as required. The function pointers get automatically populated on application start during State Machine creation. (function pvr::createNativePlatformContext). Use as a namespace (e.g. eglext::WaitSyncKHR).
glThis class contains function pointers to all OpenGL ES 2,3 and 3.1 functions. These function pointers will be populated on the initGl call, but should only be used when compatible OpenGL ES contexts are current. Use normally, using the gl class as a namespace. For example gl::BindBuffer(GL_UNIFORM_BUFFER, myBuffer);
glextContains function pointers for all OpenGL ES 2/3 extensions. The function pointers are populated when the glext::initGlext() is called (normally, PVR Shell calls this), but they must only be used when a context for which they are supported is bound.
glextRTThis class contains Function Pointers in which the OpenGL ES Ray Tracing extensions will be loaded. Extension loading happens with the glextRT::initGlextRT() function which should normally be called from the pvr::Shell. Afterwards, a pvr::native::GLESRTExtensions objects can be created and initialized with the extension string of a specific OpenGL ES context. The functions are all static and can be accessed as a namespace: glextRT::RayBounceLimit(...) etc.
HBuffer_Handle to an OpenGL ES sampler. Can be retrieved with the native_cast function on a PVRApi Sampler
HColorAttachmentView_Handle to an OpenGL ESs compiled shader program.
HColorAttachmentView_Handle to an OpenGL ES ColorAttachmentView. Dummy class for OpenGL ES implementation.
HDescriptorSet_Handle to a Descriptor Set Layout. Dummy class, as the OpenGL ES API does not have an explicit Color Attachment View object.
HPipeline_Handle to an OpenGL ES shader.
HSampler_Handle to OpenGL ES Frame-Buffer-Object.
HShader_Handle to an OpenGL ES buffer. Can be retrieved with the native_cast function on a PVRApi Buffer
