Detailed Description

This class contains function pointers to all OpenGL ES 2,3 and 3.1 functions. These function pointers will be populated on the initGl call, but should only be used when compatible OpenGL ES contexts are current. Use normally, using the gl class as a namespace. For example gl::BindBuffer(GL_UNIFORM_BUFFER, myBuffer);
