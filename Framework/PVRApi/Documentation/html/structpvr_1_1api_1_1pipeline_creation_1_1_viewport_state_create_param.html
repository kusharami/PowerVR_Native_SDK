Detailed Description

Pipeline Viewport state descriptor. Sets the base configuration of all viewports.

Defaults: Number of Viewports:1, Clip Origin: lower lef, Depth range: 0..1

scissorThe scissor rectangle of the viewport
renderSurfaceDimensionsOnly set this value in order to render to an FBO that is NOT screen sized. The default is screen size. This value must always be set to the actual dimension of the entire render surface, NOT the viewport being rendered to. It is being used to convert NDC to pixel coordinates and back, so that API-independent code can be written across OGLES and Vulkan code.
